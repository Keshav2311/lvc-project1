Instructions

1. You need to provide valid name.

2. All field are monitorred and any discrepencies will not be appreciated.

2. Provide the correct marks for intermediate.

3. The Intermediate marks will get calculated for percentage.

4. Select the appropriate option inside Metric Percentage.

5. The scholarship will be decided on below criteria:

  1. Intermediate Percentage >= 90 and Metric Percentage > 90 : 50%  Scholarship will be given.

  2. Intermediate Percentage >= 90 and Metric Percentage > 80 : 25%  Scholarship will be given.

  3. Intermediate Percentage >= 80 or <= 90 and Metric Percentage > 80 : 25%  Scholarship will be given

  4. Intermediate Percentage < 80 and Metric Percentage < 80 : No  Scholarship will be given.
